# 🚀 prj-haru (하루 / 春)

개인용 유틸리티 도구 모음 웹 앱입니다. 다양한 일상적인 계산 및 변환 기능을 제공합니다.

## ✨ 주요 기능

- **📏 단위 환산기**: 일본의 '죠(畳)' 단위를 평방미터(m²) 및 한국의 '평'으로 상호 변환
- **📅 연도/나이 계산기**: 만 나이, 세는 나이 계산 및 일본 연호 변환
- **🌓 다크/라이트 모드**: 사용자의 선호에 맞는 테마 지원
- **📱 반응형 디자인**: 모바일과 데스크톱 모두에 최적화된 대시보드 레이아웃

## 🛠 기술 스택

- **Framework**: Next.js 15+ (App Router)
- **Language**: TypeScript
- **Styling**: TailwindCSS & Vanilla CSS
- **Deployment**: Firebase App Hosting

## 🚀 시작하기

먼저 개발 서버를 실행합니다:

```bash
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 결과를 확인하세요.
